# Changelog

All notable changes to this project will be documented in this file. See [conventionalcommits.org](https://www.conventionalcommits.org/) for commit guidelines.

## [1.0.0] - 2025-03-07

### Added

- `SquircleView` — native squircle view with borderRadius, border, shadow, outline
- `SquircleButton` — pressable squircle with `activeOpacity` and render-prop children
- `buildBoxShadow` — helper for multi-layer box shadows
- `cornerSmoothing` prop (0..1) for squircle curve intensity
- `overflow` prop for content clipping
- Support for `boxShadow`, `shadowColor`/`shadowRadius`/`shadowOffset`/`shadowOpacity`, `elevation`
- Support for `outlineColor`/`outlineWidth`/`outlineOffset`/`outlineStyle`
- React Native 0.72–0.83 with New Architecture (Fabric)
- iOS and Android native implementations
